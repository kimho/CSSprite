This Project

I built this app for a few reasons. To start, I've really wanted to up my open source development stuff for my github corner of the world. I also remembered back when I did front-end stuff in school that sprites were kind of annoying to tweak and work with. Sometimes you had to refresh the page and nudge things around a bit. Bear in mind, I built this app for a demographic that is likely "front end graphic designers who do CSS but kind of not really." It isn't really ground breaking, but it was a tool that I thought would perhaps suit a few folks out there.

Current Features

  • Drag an image into the drop zone, set your elements desired width and height, then drag your sprite around to get the proper offset coordinates.
  • Accidentally stubmled across "include raw data" which allows you to encode your image right in to your CSS, if you want for whatever mobile project or what not may need that feature ;)

Upcoming
